# 相關連結
- [npm](https://www.npmjs.com/package/auto-admin-cli)
- [github](https://github.com/chiupinchun/auto-admin)

## 創建新專案
- 指令：`npx -y auto-admin-cli create <ProjectName>`，將\<ProjectName>替換為你的項目名稱。

## 初始化專案
- 指令：`npm run config`。
- 根據配置項生成多個頁面。
- **會覆蓋既有程式碼！**

## 新增頁面
- 指令：`npm run new`。
- 生成一個套好組件的頁面，並配置路由。
